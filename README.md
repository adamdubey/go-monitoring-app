# go-monitoring-app

## quick start

_via `run.sh`:_
```sh
# start docker container services
docker-compose up -d

# start ipe service
cd ipe && ./ipe

# start client app
./run.sh
# in browser, visit localhost:4000
```

_via `Makefile`:_
```sh
# start docker container services
make up

# start client app & services
make start
# in browser, visit localhost:4000
```
